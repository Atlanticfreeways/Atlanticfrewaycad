## üì∏ Document Upload Requirements

### Photo ID Guidelines
- **Format:** JPG, PNG, PDF
- **Size:** Max 10MB
- **Quality:** High resolution, all text readable
- **Lighting:** Well-lit, no glare
- **Completeness:** All four corners visible
- **Validity:** Not expired

### Proof of Address Guidelines
- **Recency:** Issued within last 3 months
- **Clarity:** All text clearly readable
- **Completeness:** Full document, not cropped
- **Name match:** Must match ID exactly
- **Address match:** Must match application

### Selfie Guidelines
- **Face visible:** Clear, front-facing
- **Lighting:** Good lighting, no shadows
- **Background:** Plain, uncluttered
- **ID visible:** Hold ID next to face
- **No filters:** Natural photo only
- **Live photo:** No screenshots
